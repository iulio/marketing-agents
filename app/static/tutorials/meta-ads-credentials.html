Meta Ads Credentials Guide

Follow these steps to get all the credentials needed to connect your Meta Ads account.

Step 1: Get Ad Account ID

From Meta Business Suite, copy the ad account ID and include the act_ prefix.

Step 2: Create a Meta App

In Meta for Developers, create a Business app and copy the App ID and App Secret.

Step 3: Get Access Token

Use Graph API Explorer to generate a user access token with ads_management permissions.

Needed:

  • App ID
  • App Secret
  • Access Token
  • Ad Account ID
