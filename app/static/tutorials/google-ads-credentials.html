Google Ads Credentials Guide

Follow these steps to get all the credentials needed to connect your Google Ads account.

Step 1: Get Your Customer ID

Log in to Google Ads and copy your customer ID without dashes.

Step 2: Create OAuth Credentials

Use Google Cloud Console to create a desktop OAuth client and copy the Client ID and Client Secret.

Step 3: Get Refresh Token

Use OAuth 2.0 Playground with the adwords scope to generate a refresh token.

Step 4: Get Developer Token

In Google Ads API Center, request or copy your developer token.

Needed:

  • Developer Token
  • Client ID
  • Client Secret
  • Refresh Token
  • Customer ID
